📰 1. 스트레이트 기사 템플릿

사용 상황: 속보 / 발표회 / 이벤트 현장 보도 / 단신

📌 설명 (인스트럭션)
- 목적: 속보/발표 내용을 역피라미드 구조로 빠르게 출고할 수 있는 기사 초안 작성. - 규칙: 5W1H 확보를 우선시할 것. 직접 인용은 발언 그대로 따옴표 안에 보존할 것. 추측·감상·논평 표현 금지. 출처 미확인 정보는 작성하지 말고 '추가 취재' 섹션에 표시할 것. 시간/수치/지명은 정확히 옮길 것.
📌 섹션 구성 (6개)
타입섹션 제목섹션 내용

GPTs Instructions한국일보 외고 소제목 생성기 (시스템 프롬프트)
당신은 한국일보 프리미엄 콘텐츠 에디터입니다. 사용자가 외부 기고문(긴 줄글)을 붙여넣으면 다음을 수행합니다: 1. 글의 논리 흐름을 분석해 3~6개 섹션으로 분리 - 각 섹션의 시작 위치(원문의 첫 문장)를 인용해서 표시 - 섹션 간 자연스러운 전환점인지 확인 2. 각 섹션마다 소제목 후보 3가지 스타일로 제안 A) 키워드형 (예: "반도체 전쟁의 서막") B) 질문형 (예: "왜 지금 반도체인가?") C) 인용/문장형 (예: "칩이 곧 국력이다") 3. 각 후보의 글자 수와 톤 분석 표시 - 글자 수: 12자 이내 권장 - 톤: 정통/비판/감성/위트 중 어느 쪽인지 4. 최종 추천 소제목 세트 (각 섹션당 1개씩 선정) - 추천 이유 한 줄 설명 - 한국일보 프리미엄 톤에 맞는지 검토 주의사항: - 자극적이거나 클릭베이트성 표현 사용 금지 - 외고 필자의 문체와 충돌하지 않도록 조율 - 영어 단어는 한자어 또는 한국어로 대체 권장 - 인명/지명 등 고유명사는 정확히 사용

  1. ChatGPT Images 2.0로 배경/메인 비주얼 생성 (한글 텍스트 포함 가능)
  2. PNG로 다운로드 (2K 권장)
  3. PowerPoint/Canva 16:9 슬라이드에 배경으로 삽입
  4. 한국일보 공식 로고 PNG를 정확한 위치에 배치 (브랜드 가이드라인 준수)

  6. 전체를 PNG/JPG로 익스포트 → 영상 송출 시스템에 업로드
💡 실무 팁: ChatGPT Images 2.0의 한글 정확도가 90%+로 올라왔지만 긴 한글 문장은 여전히 일부 깨질 수 있습니다. 핵심 키워드 1~3단어 정도는 직접 렌더로, 긴 문장이나 정확해야 하는 회사명·제목은 별도 합성 권장.

  • 기사 작성 즉시 관련 자료사진 자동 추천

Claude는 이미지+텍스트를 동시에 읽고 비교할 수 있는 AI. 그래픽 이미지와 기사 본문을 함께 첨부하면 숫자 불일치를 자동으로 찾아냅니다.

아래는 한국일보 기사 본문이고, 첨부한 이미지는 이 기사에 들어갈 그래픽입니다. [기사 본문 붙여넣기] 다음을 수행해 주세요: 1. 기사 본문에 등장하는 모든 숫자/통계/날짜를 추출 (위치 표시) 2. 그래픽 이미지에 표시된 숫자/통계/날짜를 추출 3. 두 목록을 비교해서 불일치 항목 표시 (예: 본문 12.3% vs 그래픽 12.5%) 4. 단위 차이로 인한 혼동 가능성 있는 부분 표시 (예: 만 vs 천 vs 억) 5. 그래픽 제목/범례와 본문 표현이 다른 경우 지적 6. 데스크가 확인해야 할 우선순위 (긴급/일반)

📌 데스킹 단계 권장 워크플로우

  1. 기자: 기사 + 그래픽 시안 작성
  2. 기자: Claude에 두 자료 동시 첨부
  3. 기자: 팩트체크 프롬프트 실행
  4. Claude: 불일치 항목 자동 표시
  5. 기자: 1차 수정 후 데스크 송고
  6. 데스크: 최종 사람 검토 + 출고

⚠️ 알아둘 한계

  • 이미지 텍스트 인식: 흐릿하거나 작은 글자는 오인식
  • 복잡한 차트: 막대/선/원 OK, 2축·산점도 약함
  • 사실관계 검증 별도: 숫자 일치 ≠ 사실 정확
  • AI 자체 오답: 가끔 없는 숫자를 만들어냄
